# Sprint 5 - Task Management Board

A Trello-style Task Management Board built using **React.js** that helps users efficiently manage tasks across different stages of completion.

## 🚀 Live Demo

🔗 https://sprint-5-pink.vercel.app/

## ✨ Features

- ✅ Add Task
- ✅ Edit Task
- ✅ Delete Task
- ✅ Search Tasks
- ✅ Drag & Drop (dnd-kit)
- ✅ Priority Levels (High, Medium, Low)
- ✅ Local Storage Persistence

## 🛠️ Tech Stack

- React.js
- CSS3
- dnd-kit
- Local Storage

## 📦 Installation & Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Magicalmutezzzzz/Sprint-5.git
```

2. Navigate to the project directory:

```bash
cd Sprint-5
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
