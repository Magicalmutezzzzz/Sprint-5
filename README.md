# Sprint 5 - Task Management Board

A Trello-style Task Management Board built using React.js.

#Live Demo

https://sprint-5-pink.vercel.app/

## Features
- Add Task
- Edit Task
- Delete Task
- Search Tasks
- Drag & Drop (dnd-kit)
- Priority Levels
- Local Storage

## Tech Stack
- React.js
- CSS3
- dnd-kit
- Local Storage

## Run Locally

```bash
npm install
npm run dev
